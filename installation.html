Installation

Get the PTA stack and its dependencies running before you arrive. Pick your platform.

First install Anaconda, then:

conda create -n viper -y -c conda-forge python=3.9 conda activate viper conda install -y -c conda-forge enterprise-pulsar enterprise_extensions conda install -y -c conda-forge nb_conda jupyterlab pip install la-forge pip install tqdm

On Apple-silicon Macs, build the environment under the osx-64 subdir. Install Anaconda first, then:

# build the env as osx-64 (runs under Rosetta) export CONDA_SUBDIR=osx-64 conda create -n viper -y -c conda-forge python=3.9 conda activate viper conda config --env --set subdir osx-64 conda install -y -c conda-forge enterprise-pulsar enterprise_extensions conda install -y -c conda-forge nb_conda jupyterlab pip install la-forge pip install tqdm

The PTA stack does not run natively on Windows. Install a Linux instance with WSL, then follow the Linux instructions inside it.

holodeck

The Week-1 holodeck tutorial needs a separate install. Follow the instructions at github.com/nanograv/holodeck.

Course materials

Lecture slides and tutorial notebooks live in the materials repository: github.com/stevertaylor/gw-school-2026-materials . Clone it and pull updates as the school approaches.

Stuck? There is a Software Helpdesk on the first afternoon (Mon, July 20) and instructors on hand throughout. Bring a laptop with the environment above already created.
